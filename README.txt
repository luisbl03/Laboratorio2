**INSTRUCCIONES DE COMPILACION**

- Para compilar, se usa make all. (nota: salen 2 warning, hemos intentado eliminarlos, pero no hemos podido)
- Se generan los directorios obj y exec, donde se guardan los archivos objetos y el ejecutable respectivamente

**INSTRUCCIONES DE EJECUCION**
Se han usado diferentes reglas de ejecucion, cada una para cada libro del directorio libros:

- run_leyesTrabajo
- run_leyesLiderazgo
- run_maneras
- run_sirena
- run_actitud
- run_oro
- run_influencia
- run_sueno

En estas reglas se pueden editar tanto la palabra a buscar como el numero de hilos

**INSTRUCCIONES DE LIMPIEZA**
Se usa make clean para la eliminacion de los directorios obj y exec
